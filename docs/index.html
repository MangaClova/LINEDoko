もし、家でパパの帰りを待つ子どもが、スマートスピーカーに「今どこ？」と聞けたら。

もし、LINE という使い慣れたツールから、スマートスピーカーを通して、

技術で人を幸せにしたい。そんな思いから開発に入りました。

LINE Doko の使い方

「ねぇ、Clova。LINE Doko を開いて」

お父さんやお母さん、旦那さんなど、大切なご家族に会いたくなったら。寂しい時、LINE Doko を起動しましょう

場所と伝言を伝えてくれる

LINE Doko が Clova から起動されると、スマホから登録された位置情報と伝言が伝えられます。

位置情報や伝言の登録はスマホから簡単

LINE Doko スキル連動の LINE bot から、簡単に位置情報を送信できます。

登録された情報が古い場合は、情報の更新を促す

登録された位置情報が6時間前の場合、Clova は古い情報を読み上げてから、「どこにいるかもう一度 LINE で聞いてみます」と言い、いったん終了します。

そして、Clova のオーナー(この場合はパパ)の LINE に通知を送り「パパさんが今どこにいるか気にしています」と位置情報の入力を求めます。

次にまた LINE Doko が起動された時に、この新しい位置情報が読み上げられます

無料で始めよう

  • 　 LINE Clova を持っている
  • 　 家にあなたの帰りを待つ人がいる
  • 　 スピーカーに話しかけるのが好き

以上に当てはまる人は、ぜひ始めてみましょう

  1. 【Clova】LINE Clova でスキル「LINE Doko」をインストール
  2. 【スマホの LINE】LINE bot「LINE Deko」を友達登録します。（QRコードは下)
  3. 【スマホの LINE】その友達登録した bot の指示内容に従い必要項目(パパママなど呼ばれ方)を埋めたら準備は終わり！
